X-cache Function List

Xcache Socket

  • int XcacheSocket(void)
    • Allocates a socket for communication with Xcache.

Slice manipulation

  • ChunkContext *XallocateSlice (int xcacheSock, struct sliceInfo *info)
    • Allocates XIA Cache Slice.
  • int XfreeSlice(int xcacheSock, ChunkContext *ctx)
    • Frees up XIA Cache Slice.

Content Publishing

  • int XputChunk(int xcacheSock, ChunkContext *ctx, const void *data, size_t length, ChunkInfo *info)
    • Publishes a single content chunk.
  • int XputFile(int xcacheSock, ChunkContext *ctx, const char *filename, size_t chunkSize, ChunkInfo **info)
    • Breaks file into chunks and publishes the chunks.
  • int XputBuffer(int xcacheSock, ChunkContext *ctx, const void *data, size_t length, size_t chunkSize, ChunkInfo **info)
    • Breaks buffer into chunks and publishes the chunks.

Content Retrieval

  • int XrequestChunk(int xcacheSock, sockaddr_x *addr, socklen_t addrlen, int flags)
    • Sends request to fetch content.
  • int XgetChunk(int xcacheSock, void *buf, size_t buflen, sockaddr_x *addr, socklen_t addrlen, int *flags)
    • Fetches chunk from the network and puts in buffer.

Content Access

  • int XopenChunk(const char *cid, int cidlen)
    • Opens a chunk for reading.
  • int XreadChunk(int xcacheSock, void *buf, size_t bulen)
    • Reads chunk.
  • int XgetChunkStatus(int xcacheSock, const char *cid, size_t len)
    • Fetches the current status of chunk.

Xcache Notifications

  • int XregisterNotification(int events, void (*func)(int event, const char *cid, int cidlen))
    • Registers a listener function for event(s).
  • int XprocessNotification(int xcacheSock)
    • Invokes listeneres on reception of notifications.
