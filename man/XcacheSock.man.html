NAME

XcacheSock − End point for communicating with XIA Content Caching Daemon

SYNOPSIS

#include <xcache.h>

int XcacheSocket(void);

DESCRIPTION

XcacheSock() creates an endpoint for communicating with Xcache daemon. The returned socket is already connected to Xcache.

RETURN VALUE

On success, returns file descriptor of the new, connected socket is returned. On failure, -1 is returned and errno is set appropriately.

ERRORS

XcacheSock() does not generate any errors of its own. Errors are generated by socket() call or connect() call. Errors may also be generated by underlying protocol modules.

EXAMPLE

See Xslice(8).
