NAME

XopenChunk − Open chunk for reading

SYNOPSIS

#include <xcache.h>

int XopenChunk(const char *cid, int cidlen);

DESCRIPTION

XopenChunk() creates an endpoint for communicating with Xcache daemon and associates it with chunk with content identifier as cid of length cidlen. Applications should subsequntly call XreadChunk(8) to start reading the chunk.

RETURN VALUE

On success, it returns file descriptor of the new, connected socket which is available for reading. On failure, -1 is returned and errno is set appropriately.

ERRORS

ENOTCONN

XcacheSock is not connected.

EHOSTDOWN

Xcache is not running.

EXAMPLE

See XreadChunk(8).

SEE ALSO

XreadChunk(8) Xget(8)
