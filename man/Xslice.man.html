NAME

XallocateSlice, XfreeSlice − Slice manipulation functions.

SYNOPSIS

#include <xcache.h>

ChunkContext *XallocateSlice(int xcacheSock, struct sliceInfo *info);
int XfreeSlice(int xcacheSock, ChunkContext *ctx);

DESCRIPTION

XallocateSlice() creates a context for publishing content using XputBuffer(8), XputFile(8), XputChunk(8). The argument xcacheSock must be a connected xcache socket returned by XcacheSock(). The argument info must point to a structure sliceInfo.

struct sliceInfo {

enum XcacheEvictionPolicy policy;
uint64_t size;
uint64_t ttl;
int contentStore;

};

info->policy must choose a value from following enum:

enum XcacheEvictionPolicy {

XCACHE_EVICT_POLICY_FIFO,
XCACHE_EVICT_POLICY_LRU,

};

info->size is the size of cache slice in bytes. When slice becomes full, xcache evicts a content chunk by following info->policy eviction policy. The event of eviction is notified back to the application on xcacheSock as a XCE_EVICTED notification. See XcacheNotifications(8) for more details.

info->ttl is the timeout in seconds after which all the content chunks are evicted. This eviction generates a XCACHE_SLICE_EXPIRED notification on xcacheSock. info->ttl can be set to XCACHE_INF in which case, the slice never expires until freed by XfreeSlice().

info->contentStore is unused.

XfreeSlice() frees a ChunkContext allocated by XallocateSlice(). Freeing of slice does not generate any xcache notification.

RETURN VALUE

On success, XallocateSlice() returns a chunk context. On failure, it returns NULL and errno is set appropriately. On success, XfreeSlice() returns 0. On failure, it returnes -1 and errno is set appropriately.

ERRORS

ENOTCONN

XcacheSock is not connected.

EHOSTDOWN

Xcache is not running.

EXAMPLE

See Xput(8).

SEE ALSO

XcacheSocket(8), Xput(8), Xget(8)
